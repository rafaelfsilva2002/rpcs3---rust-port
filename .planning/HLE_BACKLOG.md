# HLE wiring backlog (generated)

Generated 2026-05-29 from cross-cluster recipe discovery + adversarial verdicts (audio / net / font / imgdec).
Each "next target" row is sized to the proven playbook:
**author CC0 fixture (PSL1GHT homebrew) -> Docker build `.self` -> run via `EmuCore::run_self` -> capture NID at runtime -> add NID-dispatch match arm in `rpcs3-emu-core/src/lib.rs` -> gate green.**

Selection rules applied:
- (a) PSL1GHT-exposed (a homebrew can actually call it),
- (b) distinguishable (post-wire result differs cleanly from the all-zero unimplemented-import stub: stub logs the unresolved NID and sets `gpr[3]=0`, never writes any OUT pointer),
- (c) callback-free AND GPU/RSX/FS/net-free,
- (d) backed by an existing `rpcs3-hle-*` crate function (exception: 1 explicitly-flagged hardcode target).
